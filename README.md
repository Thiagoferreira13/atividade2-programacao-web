## Sobre o projeto
 
Desenvolvimento de uma Página Pessoal como parte da Atividade Prática 2 da disciplina de Programação Web. O objetivo foi aplicar os conceitos de estruturação HTML, estilização CSS, responsividade e JavaScript, utilizando dois frameworks CSS distintos para comparação de abordagens.

| Bootstrap | Bootstrap 5 |
